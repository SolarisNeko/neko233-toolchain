# neko233-utils-all

# Use
## maven
```xml
<dependency>
   <groupId>com.neko233</groupId>
   <artifactId>neko233-utils-all</artifactId>
   <version>0.3.2</version>
</dependency>
```

## Gradle
```kotlin
implementation("com.neko233:neko233-utils-all:0.3.2")
```
